# Simulatorbrokker
Симулятор биржи 

1.1:
MVP:
  1. Пользователь может покупать / продавать акции на внутреигровой бирже.
  2. Просматривать свой портфель акций.
  3. Просматривать историю своих сделок.
  4. Начинать новый внутригровой день.
  5. Сменить тему.
  6. Сохранить свой прогресс.

Этап 1.2:
Логические модули:
  1. core (Ядро): Сущности main.py, data_manager, chart. Валидация данных.
  2. data (Данные): Вся работа с данными JSON (создание, сохранение, загрузка операций).
  3. services (Сервисы): Логика расчета баланса, фильтрации операций.
  4. ui (Интерфейс): Окна приложения на Tkinter.
  5. application (Приложение): Сборка всех модулей воедино, точка входа.
